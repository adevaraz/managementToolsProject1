# FILTERXML Function

FILTERXML berfungsi untuk mengembalikan data tertentu dari konten XML dengan menggunakan XPath yang ditentukan.

## Sintaks

```text
FILTERXML(xml, xpath)
```

| Argumen | Fungsi |
| :--- | :--- |
| **Xml** | String dalam format XML yang valid. |
| XPath | String dalam format XPath standar. |
