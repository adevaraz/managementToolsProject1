# UNICODE Function

`UNICODE` berfungsi untuk mengembalikan angka \(titik kode\) yang terkait dengan karakter pertama dari teks.

```text
UNICODE(text)
```

| Argumen | Fungsi |
| :--- | :--- |
| text | Teks adalah karakter yang Anda inginkan nilai Unicode-nya. |
