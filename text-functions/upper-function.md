# UPPER Function

`UPPER` berfungsi untuk mengonversi teks menjadi huruf besar.

## Sintaks

```text
UPPER(text)
```

| Argumen | Fungsi |
| :--- | :--- |
| text | Teks yang ingin Anda konversi ke huruf besar. Teks dapat berupa referensi atau string teks. |
