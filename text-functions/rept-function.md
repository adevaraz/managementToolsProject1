# REPT Function

REPT berfungsi untuk mengulang teks sebanyak jumlah tertentu.

### Sintaks

```text
REPT(text, number_times)
```

| Argumen | Fungsi |
| :--- | :--- |
| text | Teks yang ingin Anda ulang. |
| number\_times | Angka positif yang menentukan berapa kali teks diulang. |
