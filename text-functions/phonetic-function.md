# PHONETIC Function

PHONETIC berfungsi untuk mengekstrak karakter fonetik \(furigana\) dari string teks.

### Sintaks

```text
PHONETIC(reference)
```

| Argumen | Fungsi |
| :--- | :--- |
| reference | String teks atau referensi ke sel tunggal atau rangkaian sel yang berisi string teks furigana. |
