# UNICHAR Function

`UNICHAR` berfungsi untuk mengembalikan karakter Unicode yang dirujuk oleh nilai numerik yang diberikan.

### Sintaks

```text
UNICHAR(number)
```

| Argumen | Fungsi |
| :--- | :--- |
| number | angka Unicode yang menyatakan karakter tersebut. |
